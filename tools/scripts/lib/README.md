# Script Libraries

Shared helpers and product-contract constants for Directive smoke verifiers, live-host probes, and soak-test scaffolding.

- `directive-contracts.mjs` centralizes dependency-free contract helpers.
- `sillytavern-live-harness.mjs` centralizes Playwright-first live SillyTavern helper utilities, artifact writers, served-extension hash checks, runtime snapshots, and safe chat-send helpers for smoke and soak tooling.
