# Canonical Interface Concepts

These four concept images are the current visual design authority for Directive:

- `starships-package-home-mobile.png`
- `mission-active-mobile.png`
- `settings-state-safety-mobile.png`
- `records-management-desktop.png`

They define the intended visual character, density, LCARS geometry, route shelf, action hierarchy, and responsive relationship between mobile and desktop. Generated wording and fictional values inside the images are not production data.

Where these concepts conflict with `docs/design/visual-targets/`, these concepts and `../DIRECTIVE_INTERFACE_DESIGN_BIBLE.md` take precedence. The older visual-target folders remain historical implementation evidence.
