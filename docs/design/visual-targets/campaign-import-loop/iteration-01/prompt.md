# Campaign Import And Library Prompt Set

## Shared Direction

Create original UX concept art for Directive's Campaign Package Library and Import Diagnostics section inside a compact SillyTavern extension shell. The target helps a player import a package archive, understand library readiness, and review the latest import result. Use an original UX-first LCARS-inspired visual identity with dark terminal negative space, asymmetric rounded rails, amber/orange command accents, lavender/blue system accents, muted red warning accents, and small modern readable typography.

Route navigation belongs in the persistent bottom command shelf. There is no top route menu requirement. The top shell is only Directive identity/status and global actions. Library and diagnostics are Campaign page-local management groups, not shell routes.

Do not sacrifice UX for LCARS aesthetics. The dropzone and Choose File browse control must remain the clear import action. Latest import diagnostics must be bounded and readable. Real controls must be visually distinct from decorative rails. No clipped labels, awkward wrapping, oversized hero text, hidden campaign facts, raw JSON, official logos, copied Star Trek screenshots, exact episode display layouts, unreadable filler text, or fake controls.
