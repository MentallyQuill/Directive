# Iteration 01 Prompts

Shared base:

Create original UX concept art for Directive's Campaign page record-management surface inside a compact SillyTavern extension shell. The target manages Creator Drafts, Saves, the current save, package library state, and Import Package. Use an original LCARS-inspired visual identity with asymmetric rounded rails, black glass surfaces, amber/coral/plum/teal accents, and crisp small modern typography. Do not use official logos, screenshots, copied Star Trek graphics, exact episode displays, unreadable filler text, or decorative fake controls.

Navigation rule:

Route navigation belongs in the persistent bottom command shelf. Do not put route navigation in the top shell. Top shell is only Directive title/status and global Back/Close actions. Records, Library, and Import may appear as page-local groups or segmented controls inside the Campaign content.

UX rule:

Do not sacrifice UX for LCARS aesthetics. Record rows must be readable, touch-safe, and easy to scan. Long names should use structured wrapping or title/meta hierarchy, not awkward truncation. No fake buttons, no clipped labels, no overlapping bottom navigation.

## Concept A

Conservative Saga-close LCARS. A compact right-side desktop extension panel showing a records status strip, Creator Drafts rows, Saves rows, a current-save highlight, and a Package Library import action. Keep density close to Saga mobile while translating rows into LCARS record tiles.

## Concept B

Rich command-console LCARS. A stronger operations-dashboard layout with a left LCARS rail, compact status dashboard, grouped save rows, creator draft rows, and subordinate import diagnostics. Multiple records remain visible without crowding.

## Concept C

Compact mobile-first LCARS and preferred target. A 390px phone shell with status tiles for Saves, Drafts, Packages, and Import, page-local Records / Library / Import grouping, a dense current-save row, two creator draft rows, and clear Load/Resume/Import actions. Bottom route navigation remains one persistent six-tab command shelf.
