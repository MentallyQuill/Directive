# Mission Side Work Visual Target Loop

Visual Target Unit: Mission Side Work, including Open Orders, Follow-Up Opportunities, scheduled side missions, active assignments, progress summaries, and the empty state.

Parent Surface: Mission page.

Primary user task: understand whether any optional or deferred operational work needs attention, then start, schedule, advance, resolve, delegate, or defer that work without losing the main mission thread.

Navigation direction: use the shared bottom command shelf as the primary route-menu model across desktop, shelf, and phone-width shells. The top shell is reserved for Directive identity/status and global Back/Close actions. Mission-local sub-tabs may live inside the Mission content because they are section controls, not route navigation.

LCARS direction: original LCARS-inspired side-work console with compact status tiles, segmented readiness rows, dense but readable assignment cards, and clear action grouping. The LCARS treatment must make optional work easier to scan and act on, not turn it into decorative chrome.

Required states:

- no side work active,
- Open Orders review candidates,
- Follow-Up Opportunity review candidates,
- scheduled follow-ups,
- active Open Orders assignment,
- active side-opportunity scene,
- completed or deferred progress interval.

Failure risks:

- side work appears as plain text after the rest of the Mission page has been redesigned,
- optional work looks more important than the main mission,
- LCARS panels crowd scene inputs or action buttons,
- long titles or reasons clip at phone width,
- bottom navigation overlaps action rows,
- route navigation drifts back into the top shell.

Acceptance focus:

- Concept C's mobile-first command shelf direction is binding.
- Side Work has an intentional empty state, not a fallback sentence.
- Active and candidate work use LCARS rows/cards with touch-safe actions.
- Bottom route navigation remains six tabs with zero top route tabs.
- Phone screenshots show no clipped labels, awkward wrapping, or overlap with the bottom shelf.
