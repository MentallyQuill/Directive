# Selected Target

Status: selected.

Selected direction: Concept C, compact mobile-first LCARS.

Binding traits:

- Bottom route navigation is the primary menu pattern across the shell.
- Top shell space is reserved for title/status and explicit Back/Close actions, not route navigation.
- Campaign content begins with package identity, readiness, and the next safe command.
- Package diagnostics and import controls are subordinate to the primary package action.
- Active route tabs stay route tabs; closing Directive remains an explicit shell action.
- LCARS geometry must improve scanability and task completion rather than acting as decoration.

Non-binding traits:

- Exact generated text.
- Exact color values.
- Decorative labels from image concepts.
- Any UI element that looks clickable without being a real control.
