# Directive 0.1.0-pre-alpha.1 Release Notes

Directive `0.1.0-pre-alpha.1` is a pre-alpha development build. It exists to prove the first campaign loop, package contract, persistence boundary, and Mission Director architecture before public alpha polish.

Last updated: June 19, 2026.

## Compatibility

- Platform target: SillyTavern extension.
- Minimum SillyTavern version: `1.12.0`.
- Runtime namespace and extension key: `directive`.
- Automatic updates are disabled with `auto_update: false`.
- Storage and package formats may change in place during pre-alpha.

## Implemented Surface

- **Directive extension identity:** `manifest.json` points to `src/extension/index.js`, registers the `directive` key, and installs a **Directive** entry in the SillyTavern extensions menu.
- **Runtime shell:** the tabbed Directive window currently exposes **Starships**, **Mission**, **Crew**, **Ship**, **Log**, and **Settings**.
- **Bundled starship package:** the Breckinridge/Ashes of Peace package is a schema-validated JSON package, not hardcoded runtime data.
- **Character Creator flow:** the Starships tab can start or resume package-owned Character Creator drafts, save partial drafts, accept the review, choose `Command` or `Exploration`, and create the first campaign save.
- **Campaign saves:** Save Game, Save As, Load Save, and stable-turn autosaves are implemented through the Directive storage repository.
- **Mission Director loop:** the Prelude mission, Chapter 1 opening handoff, first Chapter 1 response, consequence packets, narrator packets, Command Log packets, and transaction-safe state deltas are covered by deterministic tests.
- **Command Competence:** Command Briefs, Domain Reports, Request Counsel, procedural warnings, authority notes, accepted risks, and no-gotcha checks are implemented for the current Chapter 1 opening slice.
- **Command Bearing:** Inspiration and Resolve tracks can support eligible intervention prompts, two-tier outcome improvement, anchored consequences, spend recording, and transaction rollback.
- **Pressure ledger:** Prelude and Chapter 1 outcomes can seed campaign-owned pressure records and route authored Open Orders candidates without exposing hidden truth.
- **Package import normalizer:** `.directive-starship.zip` imports are normalized into package records, reject unsafe paths and active content, and report diagnostics.
- **Alpha gate:** `tools/scripts/run-alpha-gate.mjs` runs the dependency-free contract suite for the current runtime, package, storage, Director, competence, pressure, and repo-structure invariants.

## Pre-Alpha Boundaries

- Directive is not yet a public alpha release.
- Chapter 1 beyond the first response and pressure handoff is not implemented.
- Starship package import has a normalizer and diagnostics path, but not a complete player-facing import UI.
- Automatic SillyTavern message edit, delete, and branch interception remains future work; explicit runtime recovery controls exist.
- Broader host support is planned through `src/hosts/`, but the active runtime is still the SillyTavern extension shell.
- Storage migrations are intentionally not maintained during pre-alpha. Update files and docs in place to the best current version.
