# U.S.S. Celandine: Enemy's Garden

This source tree supports `directive:campaign-package:celandine-enemys-garden`.

- `campaign/`: campaign baseline, open-world contract, systems, Director reference, and endings.
- `crew/`: senior-staff character bible.
- `guardrails/`: player agency, canon, hidden-information, and failure-forward rules.
- `missions/`: tactical source records for the first three mission graphs.
- `quests/`: main quest implementation material.
- `side-missions/`: designed, standing, and emergent assignments.
- `world/`: regional gazetteer and seed-law reference.
