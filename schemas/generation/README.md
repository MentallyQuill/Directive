# Generation

Schemas for constrained generation requests and proposals.

- `continuity-projection-plan.schema.json`: utility planner fact-id and lane plan.
- `continuity-contradiction-review.schema.json`: continuity contradiction review result.
- `continuity-claim-extraction.schema.json`: generated high-risk claim extraction result.
- `continuity-projection-compression.schema.json`: compressed lower-priority projection summaries.
